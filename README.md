## 📖 About The Project

**Kids Story Studio** is an AI-powered storytelling platform designed to create personalized, educational, and engaging stories for children aged **3 to 12 years**.

Using local AI models, the application generates unique stories where the child becomes the hero of their own adventure while learning important life values and lessons.

The goal is simple:

* Make learning fun
* Encourage imagination
* Teach important values
* Provide a safe and personalized storytelling experience

## 🚀 Getting Started

### Prerequisites

* Python 3.9+
* Ollama
* Git

---

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/krishnapopat130324-art/kids-story-studio.git
cd kids-story-studio
```

### 2. Create Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Download the Model

```bash
ollama pull llama3.2
```

### 5. Run the Application

```bash
streamlit run app.py
```

## 📊 How It Works

### Step 1 — Enter Child Details

* Enter the child's name
* Select the child's age

Click the **Generate Story** button and let the AI create a unique personalized story.

* Save stories automatically
